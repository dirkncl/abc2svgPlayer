abcemb-1.js example

This XHTML document contains a ABC tune ('Au clair de la lune').

To render this tune, it includes the abc2svg core (script abc2svg-1.js) and the script abcemb-1.js which scans the document and replaces the ABC sequence(s) by SVG images.

For playing the tune, this document simply includes the script play-1.js. Check it by clicking the score!
